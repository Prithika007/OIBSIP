About the System

Our IoT-based Smart Manhole Monitoring System is designed to monitor manhole conditions in real time. Using advanced sensors, the system detects water levels, toxic gases, blockages, and unauthorized access. It sends instant alerts to municipal authorities to prevent flooding, accidents, and infrastructure damage.

Key Features

✔ Real-time water level monitoring
✔ Toxic gas detection (CH4, CO₂, H₂S)
✔ Overflow & blockage alerts
✔ GPS-enabled location tracking
✔ 24/7 monitoring dashboard
✔ Mobile alerts & cloud storage
✔ Reduces manual inspections
✔ Improves safety & reduces maintenance cost

How It Works

1. Sensors are installed inside manholes.
2. Data is sent to a cloud server via IoT communication (LoRa, Wi-Fi, GSM).
3. The system analyzes water level, gas concentration, and blockage patterns.
4. Alerts are sent to the control room or mobile devices in case of danger.
5. Dashboard displays live data, trends, and historical records.
